This paper proposes the ACDet rigid object fine-grained detector, which has the following core contributions: 1) adaptive anchor point alignment, decoupling classification centripetal deviation to achieve adaptive alignment; 2) Online tail sample supplementation algorithm to dynamically maintain category balance; 3) Adaptive grouping enhances discriminative features. Achieve SOTA with fewer resources.
